# Eggnogg+ Mirror Mode Mod
A fun little program that copies your movements in the game Eggnogg+ to P2 that I made while trying to do multiplayer.

## Dependencies
pynput - required for interacting with OS keyboard

## Instructions
1. Download mirrormode.py
2. Make sure you have dependencies
3. Open Eggnogg+ by madgarden (https://madgarden.itch.io/eggnogg)
4. Get ready to start the level
5. Click run on mirrormode.py
6. Play the "mode" if you could call it that
7. Press esc when you want it to stop
